# 🏠 Real Estate App

A modern and responsive Real Estate web application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The platform allows users to browse, search, and manage property listings with an intuitive user interface and secure authentication system.

## ✨ Features

* User Registration & Authentication
* Secure Login & Signup
* Browse Property Listings
* Search & Filter Properties
* Property Details Page
* Add, Update, and Delete Listings
* Image Upload Support
* Responsive Design
* User Profile Management
* Modern UI/UX
* RESTful API Integration

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/kfaridi1947/Real-Estate-App.git
cd Real-Estate-App
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

### Configure Environment Variables

Create a `.env` file inside the server directory:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```

If using Cloudinary:

```env
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

### Run Application

Backend:

```bash
npm run server
```

Frontend:

```bash
npm start
```
